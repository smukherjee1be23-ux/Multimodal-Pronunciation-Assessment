# Multimodal Pronunciation Assessment

## Project Description

The Multimodal Pronunciation Assessment (MPA) project is an AI-based system designed to evaluate spoken pronunciation by combining speech and text information. The system extracts acoustic and linguistic features from audio recordings and uses machine learning and deep learning techniques to assess pronunciation accuracy.

The project is implemented using Python and leverages speech processing and natural language processing technologies. It aims to provide automated pronunciation evaluation for language learning applications, improving both accuracy and consistency compared to traditional methods.

### Technologies Used
- Python
- PyTorch
- Transformers
- Wav2Vec2
- Librosa
- NumPy
- Scikit-learn

### Future Work
